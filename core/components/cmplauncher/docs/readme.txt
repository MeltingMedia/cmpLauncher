cmpLauncher
--------------------------------------------------------------------------------
Version: 1.0.0-beta1
Author: Romain Tripault // Melting Media <romain@melting-media.com>
--------------------------------------------------------------------------------

cmpLauncher allows you to display a link or redirect to a particular CMP
(Component/Extra page) based on the template ID or the resource ID you are
editing.

Feel free to suggest ideas/improvements/report bugs on GitHub:
https://github.com/MeltingMedia/cmpLauncher/issues
